# Photo-filter
### Описание проекта:
Онлайн-редактор изображений

### Особенности:
- загрузка локальных файлов в приложение
- сохранения файлов на компьютер
- Canvas API
- Fullscreen API

### Используемые технологии:
- HTML
- CSS
- JavaScript
